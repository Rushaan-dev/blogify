# Blogify

A simple blogging platform built with Node.js, Express, MongoDB, EJS, and Bootstrap.

## Features
- User authentication
- Blog creation
- Cookie-based login
- Server-side rendering with EJS

## Tech Stack
- Node.js
- Express
- MongoDB
- EJS
